# 太空帝国 MVP 上线清单

版本：v0.1.0-mvp  
日期：2026-02-22

## 1) Must Pass（上线前必须完成）

### A. 可玩性主链路
- [ 1] 新开档可正常进入游戏（国家选择可点击，无脚本报错）
- [1 ] 基础循环可运行：点击星球 -> 获得矿石 -> 建造 -> 产出资源
- [ 1] 科技系统可正常解锁（至少可从 Lv0 推进到 Lv2）
- [ 1] 阶段进度条可正常推进，文案不空白
- [1 ] 右侧“科技/伟大工程”面板可打开和关闭

### B. 槽位与建造规则
- [1 ] 星球槽位菜单可正常打开/关闭
- [ 1] 金星槽位仅允许：机器人军团、能源电站
- [1 ] 火星/木星/土星槽位按征服条件解锁（当前规则）
- [1 ] 拆除建筑不报错，资源与产量逻辑正常回算

### C. 存档稳定性
- [ 1] 自动保存正常（刷新后进度保留）
- [1 ] 读档后不出现 NaN/未定义错误
- [1 ] 连续 3 次“开始 -> 游玩 -> 刷新 -> 继续”流程无坏档
- [1 ] 点击“重新开始”可完全清档并重开

### D. 兼容与性能（最低要求）
- [1 ] 桌面 Chrome 最新版可稳定 15 分钟无致命报错
- [ 1] 手机端可点击星球、打开面板、建造（至少 1 台安卓 + 1 台 iOS）
- [1 ] 页面无持续刷屏报错（DevTools Console）

### E. 发布最小配置
- [1 ] 首页入口可访问（gameart.top 对应页面）
- [1 ] 页面标题、基础 SEO 描述存在
- [1 ] 显示版本号（建议：右下角 `v0.1.0-mvp`）
- [ 1] 提供反馈入口（邮箱或表单其一）

## 4) 上线判定（Go / No-Go）
- Go 条件：
  1. Must Pass 全部勾选；
  2. 无阻塞级 bug（无法进入、无法存档、核心功能失效）；
  3. 已有反馈入口可收集首批用户意见。

- No-Go 条件：
  1. 新开档/读档任一流程出现高概率报错；
  2. 核心循环断裂（资源不增长/无法建造/无法推进阶段）；
  3. 移动端无法进行基础操作。
